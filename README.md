Aufgabe 3:
Zuerst muss das Model trainiert werden, indem man das Programm gesture_detection_model.py ausführt. Danach kann man das media_control Skript ausführen.
Der Media controller kann 3 Dinge: einen Song pausieren, zum nächsten Song skipen und die Lautstärke erhöhen. Man pausiert mit dem Zeichen "stop" (wobei man damit auch wieder play machen kann), mit dem Zeichen "ok" kann man den Song skipen und mit dem Zeichen "one" erhöht man die Lautstärke. Die Handzeichen führt man am besten vor einer weißen Wand aus.
Mit "q" lässt sich das Programm schließen.
